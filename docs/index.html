One component

Wrap your app. Pick a mode. That's the whole API. It records silently and renders live. No backend, no accounts, no API keys. Everything stays in the browser.

import { Heatprint } from 'heatprint'

function App() {
  return (
    <Heatprint mode="aurora" opacity={0.6}>
      <YourApp />
    </Heatprint>
  )
}

The hook

For full control, use useHeatprint. Get the raw data, trigger renders manually, export PNGs. Build your own UI around it.

import { useHeatprint } from 'heatprint'

function CustomViz() {
  const {
    canvasRef,
    recording, start, stop, reset,
    download, moveCount,
  } = useHeatprint({
    mode: 'constellation',
    renderEvery: 50,
    render: { intensity: 1.5 },
  })

  return (
    <div>
      <canvas ref={canvasRef} />
      <button onClick={() => download('heatprint.png')}>
        Save ({moveCount} points)
      </button>
    </div>
  )
}

No React? No problem

Heatprint works without React. Use the tracker and renderers directly with any canvas.

import { InteractionTracker, render } from 'heatprint'

const tracker = new InteractionTracker()
tracker.start()

// Later...
const ctx = canvas.getContext('2d')
const data = tracker.getData()

render(ctx, data, { mode: 'flow' })

How it works

The tracker listens to mousemove, click, and scroll events. Coordinates get normalized to 0 through 1 so visualizations scale to any canvas size. Data stays in flat arrays with timestamps, so renderers can color by time and detect movement segments.

Each renderer interprets the data differently. The heat renderer builds a density grid and applies gaussian blur. The constellation renderer samples points and draws connections between nearby nodes. The aurora renderer fits catmull-rom splines through continuous paths and extrudes them into luminous ribbons with layered transparency.

All rendering happens on a single canvas in one requestAnimationFrame pass. No WebGL, no runtime dependencies, no external services. Under 8KB gzipped.

Export and share

Every heatprint is unique to the person who made it. Export as PNG to share, or save the raw JSON data to render later.

import { downloadPNG, exportJSON } from 'heatprint'

// Save the image
downloadPNG(data, 'heatprint.png', { mode: 'aurora' })

// Save raw data for later
const json = exportJSON(data)
localStorage.setItem('heatprint', json)
